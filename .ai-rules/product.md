# Product Vision

## What are we building?
Antigravity-SDD-Demo

This is a demonstration project for the Spec-Driven Development (SDD) methodology using the Kiro workflow within Antigravity.

## Target Audience
Developers looking to implement structured, spec-driven AI coding workflows.

## Core Features
- Structured documentation in `docs/`
- AI Rules in `.ai-rules/`
- Spec-Driven Development workflow (Planner -> Executor)
